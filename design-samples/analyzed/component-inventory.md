# Component Inventory

<!--
이 문서는 스크린샷 분석에서 도출한 investModel 초기 UI 컴포넌트 후보를 정리합니다.
프론트엔드 구현 시 컴포넌트 이름과 책임을 맞추기 위한 기준으로 사용합니다.
-->

| Component | Role | Notes |
| --- | --- | --- |
| `MobileShell` | 모바일 화면 공통 프레임 | 안전 영역, 페이지 배경, 하단 탭 기준을 포함한다. |
| `TopIconBar` | 상단 보조 액션 | 검색, 알림, 설정 같은 아이콘 버튼을 배치한다. |
| `BottomNav` | 주요 내비게이션 | 홈, 모델, 신호, 피드, 내 투자 탭을 둔다. |
| `SoftBanner` | 주요 알림/추천 | AI 시그널, 리스크 경고, 모델 추천을 부드러운 배경으로 강조한다. |
| `MetricCard` | 숫자 요약 | 예치금, 누적 수익률, 변동성, 운용 상태를 표시한다. |
| `ModelCard` | AI 모델 요약 | 모델명, 전략, 위험도, 시장, 성과, 설명 요약을 담는다. |
| `RiskBadge` | 투자 제약/위험 태그 | 레버리지, 미국장, 고위험, 채권형 등 모델 내장 성향을 표시한다. |
| `SignalRankCard` | 실시간 이슈 랭킹 | 뉴스/가격/트래픽 기반 신호를 순위형 리스트로 보여준다. |
| `FeedPost` | 피드 콘텐츠 | 모델 코멘트, 뉴스 요약, 커뮤니티형 반응을 담는다. |
| `ModelDetailMandate` | 모델 운용 원칙 | 사용자가 바꿀 수 없는 모델의 투자 성향과 제한을 설명한다. |
| `DisclosurePanel` | 위험 고지 | 성과 비보장, 손실 가능성, AI 판단 한계 등을 노출한다. |
| `SectionHeader` | 섹션 제목 | 제목, 설명, 우측 액션을 일관되게 배치한다. |
| `ChipTab` | 필터/세그먼트 | 모델 유형, 시장, 신호 소스 전환에 사용한다. |
| `FloatingActionButton` | 주요 단일 액션 | 모델 선택, 예치금 추가 같은 핵심 액션에만 제한적으로 사용한다. |
