## Brand Tone

여기에 브랜드 톤을 입력해주세요.

## Target Impression

여기에 사용자가 앱을 처음 봤을 때 느껴야 할 인상을 입력해주세요.

## Observed Layout Patterns

아직 스크린샷이 제공되지 않았습니다. 화면 캡처를 제공하면 `visual-seed-from-screenshot` 스킬이 관찰 가능한 레이아웃 패턴을 채웁니다.

## Color Palette

여기에 사용할 색상 원칙을 입력해주세요.

## Typography

여기에 폰트, 크기, 굵기, 행간 기준을 입력해주세요.

## Components

초기 예상 컴포넌트:

- 모델 카드
- 모델 상세 페이지
- 리스크 배지
- 수익률/변동성 차트
- 자금 입금 상태 패널
- 운용 로그 타임라인
- 운영자 심사 테이블

## Do Not

- 수익을 보장하는 듯한 시각 표현을 사용하지 않는다.
- 위험 정보를 작게 숨기지 않는다.
- 레버리지/고위험 모델을 게임처럼 보이게 만들지 않는다.
