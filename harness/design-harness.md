## Brand Tone

여기에 사용자가 원하는 브랜드 톤을 입력해주세요.

임시 기준:

- 과장된 수익 약속보다 신뢰, 검증, 투명성을 우선한다.
- AI 모델 마켓플레이스처럼 탐색이 쉬워야 하지만, 투자 앱답게 위험 고지를 명확히 보여준다.
- 전문적이되 차갑지 않고, 초보자도 정보 구조를 따라갈 수 있게 한다.

## Target Impression

여기에 사용자가 앱을 처음 볼 때 느끼길 원하는 첫인상을 입력해주세요.

임시 기준:

- “AI 모델을 골라 투자 전략을 위임하는 앱”이라는 핵심이 첫 화면에서 드러난다.
- 돈, 모델 성과, 위험 등급, 운용 범위가 빠르게 비교된다.
- 게임처럼 보이지 않고, 금융 서비스의 조심스러운 밀도를 유지한다.

## Observed Layout Patterns

`design-samples/selected/`의 모바일 캡처 20장을 기준으로 관찰한 패턴:

- 1170 x 2532 비율의 세로형 모바일 화면이며, 앱 UI 기준으로는 약 390 x 844 프레임에 대응된다.
- 큰 페이지 타이틀, 우측 상단 아이콘 버튼, 하단 탭 내비게이션이 반복된다.
- 화면 배경은 대부분 흰색 또는 아주 옅은 회색/블루 톤이고, 정보 묶음은 둥근 카드와 리스트 행으로 정리된다.
- 주요 카드에는 넓은 여백, 20px 전후의 큰 라운드, 미묘한 그림자 또는 구분선이 사용된다.
- 실시간 이슈/랭킹 화면은 파란 그라데이션 헤더와 흰색 카드 리스트를 조합한다.
- AI 소개 화면은 옅은 블루 배경 위에 큰 제목과 기능 카드가 세로로 쌓인다.
- 피드 화면은 소셜 포스트 카드, 태그, 이미지형 콘텐츠, 반응 버튼을 사용한다.
- 계좌/홈 화면은 시장 지수, 요약 배너, 계좌 카드, 자산 행, 하단 탭을 한 화면에 압축한다.

## Color Palette

관찰 기반 후보 색상이며, 최종 브랜드 컬러는 사용자가 확정해야 한다.

- 배경: `#FFFFFF`, `#F7F9FC`, `#F2F6FF`
- 표면: `#FFFFFF`, `#F4F6FA`
- 구분선: `#E9EEF5`
- 주요 액션/강조: `#2F80ED`, `#4C8DFF`
- 상승/위험 강조: `#F04452`
- 하락/보조 강조: `#3182F6`
- 본문 텍스트: `#202632`
- 보조 텍스트: `#7D8796`
- 비활성 텍스트: `#A5ADBA`

## Typography

폰트는 아직 확정하지 않는다. 초기 Figma 구조에서는 `Inter`를 사용하고, 실제 서비스에서는 한국어 가독성이 좋은 폰트로 교체한다.

- 화면 타이틀: 30-36px, Bold
- 섹션 제목: 20-24px, Bold 또는 Semi Bold
- 카드 제목: 16-18px, Semi Bold
- 본문/리스트: 14-16px, Regular 또는 Semi Bold
- 보조 설명/배지: 11-13px, Medium
- 숫자/수익률: 명확한 굵기 대비를 주되 과도한 장식은 피한다.

## Components

초기 예상 컴포넌트:

- `MobileShell`: 모바일 화면 프레임, 안전 영역, 하단 탭 포함
- `TopIconBar`: 검색, 알림, 설정 등 상단 아이콘 영역
- `BottomNav`: 홈, 모델, 신호, 피드, 내 투자 탭
- `SoftBanner`: AI 시그널/공지/리스크 안내 배너
- `MetricCard`: 예치금, 누적 수익률, 변동성, 모델 운용 상태 카드
- `ModelCard`: AI 모델명, 운용 범위, 위험도, 누적 성과, 설명 요약
- `RiskBadge`: 레버리지, 미국장, 채권형, 고위험 등 모델 전략 태그
- `SignalRankCard`: 뉴스/가격/트래픽 기반 이슈 랭킹 카드
- `FeedPost`: 모델 운용 코멘트, 뉴스 요약, 커뮤니티형 피드 카드
- `ModelDetailMandate`: 모델이 정한 투자 원칙, 허용 자산, 제한 사항 설명 블록
- `DisclosurePanel`: 투자 위험, 성과 비보장, 모델 책임 범위 안내

## Do Not

- 수익을 보장하는 듯한 시각 표현이나 문구를 사용하지 않는다.
- 위험 정보, 수수료, 모델 운용 범위, 투자 제한 사항을 작게 숨기지 않는다.
- 레버리지/고위험 모델을 게임처럼 보이게 만들지 않는다.
- 참고 스크린샷의 고유 브랜드, 로고, 문구, 구체적인 화면 구성을 그대로 복제하지 않는다.
- 사용자가 정해야 하는 브랜드 철학, 카피, 로고, 최종 컬러를 임의로 확정하지 않는다.
